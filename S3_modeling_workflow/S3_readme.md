Supplementary Data S3: Redacted 201-bus model-preparation workflow

This folder provides the redacted task prompt and generated workflow plan for the 201-bus model-preparation case discussed in Section VI.

The purpose of this supplementary material is to document how the high-level modeling intent was converted into a structured, engineer-supervised model-preparation workflow. 

The files included in this folder are:

S3_task_prompt.txt: the redacted task prompt used for the model-preparation workflow;
S3_workflow_plan.json: the generated workflow plan, including task decomposition, clarification items, and engineer-review points.

The original regional grid model contains confidential topology, component names, electrical parameters, and operating data. Therefore, the raw model file, original component names, detailed topology, electrical parameters, and source engineering data are not publicly released.

This supplementary material is not intended to reproduce the confidential 201-bus grid model. Instead, it provides a traceable workflow-level record showing how the proposed LLM-enabled framework organizes the model-preparation process and identifies items requiring engineer confirmation.
